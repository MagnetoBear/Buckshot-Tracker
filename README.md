This is a buckshot tracker to track the rounds left inside the shotgun in the game Buckshot Roulette. It gives you the odds of the next bullet and helps you save £3 on the Roblox version.

Includes an HTML, Python, and EXE version.

IMPORTANT:
If you don't have experience running code, such as HTML or Python scripts, or want to use the app quickly, use the exe file.

If you do have experience in running scripts, it should be pretty self-explanatory

Python dependencies: pywebview
For Python, run:
pip install pywebview
python buckshot_tracker.py
